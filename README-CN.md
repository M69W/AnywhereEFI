# AnywhereEFI



[中文版](https://github.com/penghubingzhou/AnywhereEFI/blob/master/README-CN.md)|[English](https://github.com/penghubingzhou/AnywhereEFI/blob/master/README.md)



本脚本是一个给黑苹果使用的自动化EFI工具 (尤其是对于四叶草EFI).





# 特性



- 自动安装EFI

- 在安装时自动挂载EFI分区

- 自动更新Clover EFI版本

- 以及未来更多新特性

  

尽情享受它吧!
